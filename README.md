# 🏠 Property Rent Predictor

A Machine Learning web application that predicts the estimated monthly rent of a property based on its size.

The application is built using Python, Scikit-Learn, and Gradio, and is deployed on Render for easy access through a web browser.

---

## 📖 About The Project

Property Rent Predictor is a simple Machine Learning project that estimates rental prices using a trained Linear Regression model.

Users can enter the size of a property, and the application predicts the expected monthly rent instantly through an interactive web interface.

This project demonstrates the complete machine learning workflow from model training to deployment.

---

## 🚀 Features

- Predict property rent using Machine Learning
- Simple and interactive user interface
- Real-time prediction results
- Model deployed on the cloud using Render
- Easy-to-use web application

## ⚙️ How It Works

1. User enters the property size.
2. The trained machine learning model receives the input.
3. The model predicts the estimated rent.
4. The predicted rent is displayed instantly on the screen.
